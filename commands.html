moderation

ban @ | bans a user

kick @ | kicks a user

mute @ | mutes a user

cmute @ [time] | mutes a user in the dedicated channel for a certain amount of time

botclean [Amount] | Deletes the dedicated amount of messages send by Blood

Miscellaneous

afk [Reason] | Sets the user as AFK and display an AFK message when mentioned

ping | shows the latency of the bot

verify | Initiate the verify process

setmod | setup moderation (makes mod-log channel and imute, rmute and muted role)
